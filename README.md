# Code from Front-End Project Week Solution Lecture

## How to Use:

- Install dependencies in `client` AND `server` directories:

```
// While in root directory
cd client
npm i (or yarn)
```

```
// While in root directory
cd server
npm i (or yarn)
```

- Start the `server`

```
// While in root directory
cd server
npm start
```

- Start React app
```
// While in root directory
cd client
npm start
```
